## Sobre o Projeto

O **TripScripter** é uma plataforma desenvolvida para auxiliar usuários no planejamento e organização de viagens de forma simples e intuitiva. O sistema foi criado como parte do Projeto Integrador do curso de Análise e Desenvolvimento de Sistemas do CEUB, com o objetivo de centralizar informações importantes da viagem em um único ambiente

A aplicação permite que os usuários criem roteiros personalizados, organizem atividades por dia, visualizem locais em mapas interativos e mantenham anotações importantes para cada etapa da viagem

## Funcionalidades

### ✈️ Criação de Roteiros
- Cadastro e gerenciamento de múltiplas viagens
- Definição de datas de início e término

### 📅 Planejamento Diário
- Organização das atividades por dia
- Controle visual do itinerário da viagem

### 🗺️ Mapas Interativos
- Visualização de locais e pontos de interesse.
- Navegação facilitada através da integração com mapas

### 📝 Bloco de Notas Integrado
- Registro de observações e lembretes
- Armazenamento de informações relevantes para cada atividade

## Tecnologias Utilizadas

- HTML5 – Estrutura da aplicação
- CSS3 – Estilização e responsividade
- JavaScript (Vanilla) – Lógica de funcionamento do sistema
- Leaflet.js – Implementação dos mapas interativos
